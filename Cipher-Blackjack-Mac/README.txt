# Cipher Blackjack

## How to Install:
1. Unzip this file
2. Drag "Cipher Blackjack.app" to your Applications folder
3. Double-click to play!

## First Launch:
macOS may show a security warning for unsigned apps.
If this happens:
1. Right-click the app
2. Select "Open" 
3. Click "Open" in the dialog

## System Requirements:
- macOS 10.14 or later
- 100MB free space

Enjoy the game!
